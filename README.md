# 🏙️ CityFix – Civic Issue Reporting System

## 📌 Overview

CityFix is a web-based civic issue reporting platform that enables citizens to report public infrastructure problems such as potholes, streetlight failures, garbage accumulation, water leakage, and other municipal issues. The system helps bridge the gap between citizens and local authorities by providing an easy-to-use complaint management platform.

---

## 🚀 Features

* 📝 Register and submit civic complaints
* 📍 Report issues with title, description, and category
* 📊 View submitted complaints
* 🔄 Track complaint status
* 🏙️ User-friendly interface
* 📱 Responsive design
* 🌐 Multi-language support (upcoming feature)

## ⚙️ Installation

### Clone the Repository

```bash
git clone https://github.com/YOUR_USERNAME/CityFix.git
cd CityFix
```

---

## Frontend Setup

```bash
cd frontend
npm install
npm start
```

Frontend runs on:

```text
http://localhost:3000
```

---

## Backend Setup

```bash
cd backend
npm install
node server.js
```

Backend runs on:

```text
http://localhost:5000
```
